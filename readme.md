# FIAS API

Инструмент для поиска по [федеральной информационной адресной системе](https://fias.nalog.ru) Российской Федерации. 

Технологии и библиотеки, используемые инструментом:
- Web
    - apache
    - php >= 7.1.3
    - php-curl
    - php-dom
    - php-json
    - php-libxml
    - php-mysqli
    - php-simplexml
    - php-soap
    - php-xmlreader
    - fideloper/proxy
    - folklore/graphql
    - kebacorp/vaultsecret
    - laravel/framework
    - laravel/passport
    - laravel/tinker
    - sngrl/sphinxsearch
    - inok/dbf
- Sphinxsearch
    - sphinxsearch engine >= 3.0
    - nginx
- MariaDB >= 10.4
